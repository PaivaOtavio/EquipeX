<h2 align="center"> Como instalar </h2>

Você precisa ter o nodejs na última versão LTS e o NPM (node package manager). 

> Clone o projeto
```bash
git clone https://github.com/PaivaOtavio/ProjetoConnexio.git
```
> Instale as dependências
```bash
npm install 
```

Rode o projeto
```bash
npm run serve
```
